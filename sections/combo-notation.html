This quick reference explains how combos are written on the page so you can read each route at a glance. The notation mirrors the standard Dustloop numeric system and adds a few Kokonoe-specific shorthand terms.

Directions (1-9)
Use the numpad layout to indicate stick directions. For example, 6A is a forward A normal and 22B represents a down-down command normal.
Button Prefixes
j. denotes the move is performed in the air, while dj. means after a double jump. Charged moves add a c. prefix, such as c.214A.
Special Resources
Graviton placements appear as Grav(#), where the number indicates the stock used. Fireball set-ups use Fireball(#) to show the bomb timer and Superball highlights the staff special.
Enders and Links
Tilde ~ shows rapid cancel links, while commas group normal gatlings. Enders such as 236236C or j.214B identify the knockdown state generated.

If you add new combos or explanations, keep the same shorthand so the automatic formatting highlights every token correctly. Anything written with the notation above, like 3C > 22B > j.B, will be color coded automatically.
